# 인싸전 - 당신을 위한 신조어 사전 📖

**인싸전**은 **'인싸 + 사전'** 의 합성어로 한글 신조어 사전입니다.

**1️⃣ 주요 기능** 

  - 문장 크롤링
  
  - 데이터 전처리
  
  - 신조어 추출 
  
  - 카테고리 분석
  
  - 유사 단어 분석
  
  - 감성 분석

---

  
**2️⃣ 기술 스택**

  - 주언어 : Python 
  
  - 신조어 추출 : Python selenium, beautiful soup 
  
  - 형태소 분석 : Konlpy - okt, komoran / Soynlp - MaxScoreTokenizer
  
  - 모델 생성 : tensorflow 프레임워크
  
  - 웹서버 : AWS EC2, Flask 
  
  - 감성 사전 : KNU 감성 사전 
  
  - 확장 프로그램 : html, css, javaScript
